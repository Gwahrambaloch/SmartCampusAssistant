A comprehensive student assistant application built with Python and Streamlit.

## Features
- **Task Manager**: Track assignments and deadlines.
- **Attendance Tracker**: Visual attendance stats.
- **Grade Calculator**: Estimate your GPA.
- **AI Assistant**: Get quick answers to campus queries.

## Setup & Run

1.  **Install Requirements**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the App**:
    ```bash
    streamlit run app.py
    ```

3.  **Login**:
    -   Register a new account on the first screen.
    -   Login to access the dashboard.
